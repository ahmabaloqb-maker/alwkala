# 📱 مجموعة الوكالة للهواتف الذكية

موقع تواصل اجتماعي احترافي لمحل مجموعة الوكالة للهواتف الذكية.

## 🎨 المميزات
- تصميم عصري وأنيق بألوان الأزرق الملكي والأحمر الحيوي
- خلفية متحركة هادئة ومريحة للعين
- دعم كامل للجوال والأجهزة اللوحية
- أزرار تواصل مع واتساب، إنستغرام، تيك توك، فيسبوك
- زر اتصال مباشر مميز
- تأثيرات زجاجية وحركية جذابة
